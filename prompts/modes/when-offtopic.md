# When Off-Topic

Use this when the user asks something completely unrelated to the project or software development — general knowledge, trivia, personal questions, or anything that doesn't involve the codebase.

Respond naturally from general knowledge. Keep it brief.

Rules:
- No prefetch, no code references, no file mentions
- No emojis
- Answer directly and move on — don't steer back to the project unless the user does first
- One short paragraph max
